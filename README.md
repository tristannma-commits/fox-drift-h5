# 狐狸的奇幻漂流日记 · H5 交互原型

QQ炫舞手游夏日航行《狐狸的奇幻漂流日记》活动 H5 可交互原型，供设计同学做交互参考。

## 在线预览 / 本地运行

直接用浏览器打开 `index.html` 即可，或本地起一个静态服务器：

```bash
# Python
python -m http.server 8011
# 然后访问 http://localhost:8011/index.html
```

> 推荐用静态服务器打开（部分浏览器对 `file://` 下的字体/canvas 有限制）。

## 设计主题：夜航寻缘

深夜海洋 + 暖橙狐狸灯光的氛围，契合"狐狸摆渡人 / 月老 + 夏日漂流"母题。

- **配色**：深靛海蓝打底，月光金 `#f4d79a`、狐狸橙 `#ff9248`、瓶光青 `#82e6d6`
- **字体**：Noto Serif SC + Fraunces italic + Ma Shan Zheng 手写体
- **氛围**：Canvas 实时渲染月亮/月光倒影、星空、三层波浪海面、漂浮发光漂流瓶 + grain 噪点
- **主角**：纯 SVG 手绘狐狸坐月牙小船，呼吸浮动，捞瓶时潜水下沉

## 交互链路

1. **海洋 · 捞瓶**：性别筛选 → 盲盒打捞动画 → 随机陌生人漂流瓶卡片
2. **三层互动**：轻互动卡片（匿名可用）→ 实名留言（带 UID）→ 回响箱
3. **端内收口**：实名瓶可复制 UID 去加好友
4. **投瓶**：选模板 → 写文案 → 匿名开关 → ACE 审核动画 → 起航
5. **航行日记 / 回响箱**：我的瓶被捞数据回响

## 元素隐喻体系

| 隐喻符号 | 真实含义 |
|---|---|
| 狐狸 | 摆渡人 / 月老（捞瓶、递瓶、带回响的人格化） |
| 船 | 投放=出海 / 被捞=靠岸 的动作载体 |
| 漂流瓶 | 展示模板 + 交友文案（社交名片） |
| 海洋 | 陌生人社交池 |
| 回响箱 | 缘分消息容器 |
